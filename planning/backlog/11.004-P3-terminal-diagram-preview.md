# Story: Terminal Diagram Preview

**ID**: 11.004
**Epic**: 11 — Diagram Rendering
**Priority**: P3
**Status**: backlog

## Goal

Show diagram preview in the TUI note pane (stretch goal).

## Options

1. ASCII art approximation (simple, universal)
2. Sixel protocol (iTerm, foot, WezTerm — actual image in terminal)
3. Kitty graphics protocol (kitty terminal)

## Exit Criteria

- [ ] Diagram blocks show a visual indicator in the note pane
- [ ] At minimum: "[Diagram: mermaid]" placeholder with distinct styling
- [ ] Stretch: sixel/kitty inline image rendering

## Notes

- Lower priority than docx embedding
- Sixel support via `sixel-rs` or raw escape sequences
- Could cache rendered PNGs and display on demand
