# Story: Markdown Import

**ID**: 01.005
**Epic**: 01 — Core Data Model and Storage
**Priority**: P1
**Status**: ondeck

## Goal

Import a markdown document as a todo subtree.

## Exit Criteria

- [ ] Headings (`#`, `##`, etc.) become tree nodes at corresponding depth
- [ ] Checkbox items (`- [ ]`, `- [x]`) become leaf items with completion state
- [ ] Text between headings becomes the note for that heading's node
- [ ] Nested lists map to nested items
- [ ] Round-trip: export then import produces equivalent tree
- [ ] Unit tests

## Tasks

- [ ] 01.005.01 — Implement markdown import parser
- [ ] 01.005.02 — Unit tests including round-trip

## Notes

- Heuristic parser — won't handle arbitrary markdown perfectly
- Focus on structured task documents (headings + checklists)
