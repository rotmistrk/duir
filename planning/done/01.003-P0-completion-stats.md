# Story: Completion Percentage Computation

**ID**: 01.003
**Epic**: 01 — Core Data Model and Storage
**Priority**: P0
**Status**: ondeck

## Goal

Recursively compute completion stats for tree nodes: leaf counts,
checked counts, percentage, and auto-derive parent completion state.

## Exit Criteria

- [ ] Leaf items: completion is their own checked state
- [ ] Branch items: completion derived from children (all done=Done, some=Partial, none=Open)
- [ ] Percentage computed as checked_leaves / total_leaves * 100
- [ ] Stats recomputed after any tree mutation
- [ ] Unit tests with multi-level trees
