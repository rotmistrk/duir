**ID**: 01.007
**Epic**: 01 — Core Data Model and Storage
**Priority**: P2

## Goal

Import legacy `.todo` XML files from the Qt ToDo app, converting HTML notes to markdown.

## Exit Criteria

- [ ] Parse `todo-tree` XML format (version 1.0 and 1.1)
- [ ] Map item attributes (title, folded, important, completed) to model
- [ ] Convert HTML rich-text notes to markdown
- [ ] Output a valid TodoFile
- [ ] Unit test with sample `.todo` file
