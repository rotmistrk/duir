# Story: Tree Operations

**ID**: 01.002
**Epic**: 01 — Core Data Model and Storage
**Priority**: P0
**Status**: ondeck

## Goal

Implement all tree manipulation operations on the model: add, delete, move,
clone, sort, promote, demote, swap.

## Exit Criteria

- [ ] Add sibling/child item at any position
- [ ] Delete item (with children)
- [ ] Clone subtree
- [ ] Swap item with previous/next sibling
- [ ] Promote item (move to parent's level)
- [ ] Demote item (make child of preceding sibling)
- [ ] Sort children alphabetically
- [ ] Move item between files (cut from one tree, paste into another)
- [ ] All operations have unit tests

## Tasks

- [ ] 01.002.01 — Implement tree path addressing
- [ ] 01.002.02 — Implement add/delete/clone operations
- [ ] 01.002.03 — Implement move operations (swap, promote, demote)
- [ ] 01.002.04 — Implement cross-file move
- [ ] 01.002.05 — Unit tests

## Notes

- Operations work on the model directly (no UI dependency)
- Tree path: Vec<usize> addressing item by index at each level
- Cross-file move = remove from source + insert into target
