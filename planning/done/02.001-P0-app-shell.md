# Story: App Shell, Layout, and Tree Rendering

**ID**: 02.001
**Epic**: 02 — Terminal UI
**Priority**: P0
**Status**: ondeck

## Goal

Build the ratatui app shell with split layout, tree rendering with all visual
attributes, and basic keyboard navigation. This is the foundation for all TUI work.

## Exit Criteria

- [ ] App starts, renders layout, exits cleanly on 'q'
- [ ] Left pane: tree with indentation, checkboxes, completion %, importance
- [ ] Right pane: note display area (placeholder)
- [ ] Bottom: status bar with key hints
- [ ] Tree navigation: up/down/left(collapse)/right(expand)
- [ ] Multi-file: top-level nodes are files
- [ ] Visual: bold for important, strikethrough for completed
- [ ] Loads .todo.json files from a directory on startup

- [ ] 02.001.01 — App struct, event loop, layout
