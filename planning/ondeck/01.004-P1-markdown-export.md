# Story: Markdown Export

**ID**: 01.004
**Epic**: 01 — Core Data Model and Storage
**Priority**: P1
**Status**: ondeck

## Goal

Export a subtree (or full file) as a markdown document.

## Exit Criteria

- [ ] Top-level selected item maps to `#` heading
- [ ] Nested items map to heading levels or checkbox lists
- [ ] Notes rendered as markdown paragraphs under their item
- [ ] Important items marked with emphasis
- [ ] Completed items use `[x]`, open use `[ ]`
- [ ] Output is valid, readable markdown
- [ ] Unit tests

## Tasks

- [ ] 01.004.01 — Implement markdown export
- [ ] 01.004.02 — Unit tests

## Notes

- Configurable depth cutoff for headings vs checkbox lists
- Default: depth 1-3 as headings, deeper as nested checkbox lists
