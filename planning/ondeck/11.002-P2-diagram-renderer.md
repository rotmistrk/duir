# Story: External Renderer Integration

**ID**: 11.002
**Epic**: 11 — Diagram Rendering
**Priority**: P2
**Status**: ondeck

## Goal

Render diagram source to PNG via external tools (mermaid-cli, plantuml, graphviz).

## Exit Criteria

- [ ] Render mermaid diagrams via `mmdc` (mermaid-cli)
- [ ] Render plantuml diagrams via `plantuml` (Java)
- [ ] Render dot/graphviz diagrams via `dot`
- [ ] Graceful fallback if tool not installed (return error, don't crash)
- [ ] Configurable tool paths in config.toml
- [ ] Temp file management (create, render, clean up)
- [ ] Unit tests with mock (test the pipeline, not the external tool)

- [ ] 11.002.01 — DiagramRenderer trait and implementations per engine
- [ ] 11.002.02 — Config for tool paths
- [ ] 11.002.03 — Temp file pipeline: write source → invoke tool → read PNG → clean up
