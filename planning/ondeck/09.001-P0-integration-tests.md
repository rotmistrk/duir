# Story: Comprehensive Integration Tests

**ID**: 09.001
**Epic**: 09 — Testing
**Priority**: P0
**Status**: ondeck

## Goal

Add integration tests for tree operations that verify data integrity
after mutations, especially around cloning, moving, and path consistency.

## Exit Criteria

- [ ] Clone subtree preserves all data, original unchanged
- [ ] Clone + navigate: correct item at each path
- [ ] Add sibling/child: paths of existing items remain valid
- [ ] Delete: remaining items have correct content
- [ ] Move (promote/demote/swap): content follows the item
- [ ] Encrypt/decrypt round-trip preserves all data
- [ ] Collapse/expand markdown round-trip preserves data
- [ ] Filter does not corrupt data
